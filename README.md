# 🚗 Road Accident Analysis Dashboard (Power BI)

## 📊 Project Overview
This project analyzes road accident data to uncover key insights related to accident trends, severity, location, and contributing factors. The dashboard is built using Power BI and helps stakeholders make data-driven decisions to improve road safety.

---

## 🎯 Objectives
- Analyze accident trends over time
- Identify high-risk locations
- Understand factors contributing to accidents
- Evaluate severity distribution
- Provide actionable insights for safety improvement

---

## 🛠️ Tools & Technologies
- Power BI (Data Visualization)
- Excel (Data Source)
- DAX (Data Analysis Expressions)
- Data Cleaning & Transformation

---

## 📂 Dataset
- Source: Road Accident Data (Excel)
- Contains:
  - Order_ID / Accident_ID
  - Date & Time
  - Location (City/Region)
  - Vehicle Type
  - Severity Level
  - Weather Conditions
  - Casualties

---

## 📈 Key KPIs
- Total Accidents
- Total Casualties
- Severity Distribution (Fatal, Serious, Minor)
- Monthly Accident Trends
- Accidents by Location
- Accidents by Weather Conditions

---

## 📊 Dashboard Features
- 📅 Time-based analysis (Monthly trends)
- 📍 Location-based insights
- 🚗 Vehicle type impact analysis
- 🌧️ Weather condition analysis
- ⚠️ Severity breakdown

---

## 🔍 Key Insights
- Most accidents occur during peak traffic hours
- Certain locations have consistently higher accident rates
- Weather conditions like rain increase accident probability
- Majority of accidents fall under minor severity
